Iris Authentication for Secure Voting System

Get Started


Abstract

This project introduces a robust iris recognition system tailored specifically for enhancing the security and reliability of the voting process. Traditional voting systems often encounter issues like dummy votes and multiple registrations, compromising the integrity of the electoral process. To address these challenges, we propose a deep learning-based iris recognition approach, leveraging convolutional neural networks (CNNs) for feature extraction and recognition.
